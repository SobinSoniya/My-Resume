An Enhanced Digital Signature based Routing Protocol in WSN

The aim of this project is to improve the security in Wireless Sensor Network. Whereas the previous Digital Signature schemes are do not provide better security. Here we used 128 bit Digital Signature algorithm to improve the security. Properly programmed digital signatures are more difficult to forge than other type of security measures. This system also shows efficiency in packet delivery fraction and energy consumption.

An Efficient Algorithm for Intrusion Detection in a Network

The objective of this project is to detect the intruder in a network. Here we used two algorithms for detecting the intruder. They are Conditional Random Field Approach and Layered Approach. If any intruder cross the client and server communication these two algorithms are used to find that intruder and produce alarm sound.
